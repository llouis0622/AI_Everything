# 1. 극한(limit)

- 영원히 가까워지지만 결코 도달하지 못하는 값을 표현하는 방법
    
    $$
    \lim_{x \rarr a}f(x) = C
    $$
    
- x가 무한대에 가까워짐에 따라 함수 1/x는 0에 가까워짐
    
    $$
    \lim_{n \rarr \infty}(1 + \frac{1}{n})^n = e = 2.71828169413...
    $$
    
- 좌극한
    
    $$
    \lim_{x \rarr a - 0}f(x) = C
    $$
    
- 우극한
    
    $$
    \lim_{x \rarr a + 0}f(x) = C
    $$
    
- 함수의 극한 값 존재 = 함수의 좌극한 값과 우극한 값이 동일
